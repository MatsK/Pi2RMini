Pi2RMini 
2013-2014 Logan Microelectronics

Small power switch for the Raspberry Pi. Free to use, modify, and distribute with proper attribution. This directory holds the current schematic and layout (Eagle v6.1.0), single layer PDF (not mirrored), code for ATTiny13 and Raspberry Pi, and older versions.
